# Prompt Iteration Examples
## Initial baseline prompt 
Prompt : Generate a caption for this image

### Observed issues 
- Captions were often generic 
- No clear tone or personality 
- Inconsistent output length 
- Limited relevance to user intent 

## Iteration 1 - Tone guidance 
Prompt: Generate a short caption for this image with a positive tone

### Improvement
- More engaging language
- Slightly improved relevance 
- Still inconsistent formatting 

## Iteration 2 - Structured guidance 
Prompt: 
Generate a short, engaging social media caption for this image. Use a friendly tone and keep the output concise

## Improvement 
- More consistent output
- Better alignment with social media usage 
- Reduced verbosity 

## Final Prompt Strategy 
The final approach uses a structured prompt that
- Describes the task clearly 
- Injects user selected tone or vibe
- Avoids strict formatting rules

### Result 
This strategy consistently produced captions that felt natural, context aware and suitable for direct use in the application 

## Summary 
Prompt iteration was essential to improving output quality. Small refinements had a noticeable impact, reinforcing the importance of experimentation and evaluation when working with Generative AI systems
